# Entre Sábios

Website estático do projeto **Entre Sábios**.

O site reúne reflexões filosóficas, contos filosóficos, ensaios editoriais e recomendações de leitura conforme sentimentos selecionados pelo usuário.

## Estrutura principal

- `index.html` — página principal do website.
- `style.css` e `css/` — estilos da interface principal.
- `script.js` e `js/` — dados, algoritmo de sentimentos e funcionalidades do site.
- `seo.css` — estilos das páginas indexáveis.
- `sentimentos/`, `contos/`, `pensadores/`, `ensaios/` — páginas SEO geradas.
- `assets/` — imagens e ilustrações usadas pelo site.
- `sitemap.xml` e `robots.txt` — arquivos para rastreamento por mecanismos de busca.

## Publicação

O conteúdo desta pasta corresponde ao pacote pronto para hospedagem.

Para publicar na Hostinger, envie os arquivos desta pasta para o diretório público do domínio.

Domínio principal: <https://www.entresabios.com/>
