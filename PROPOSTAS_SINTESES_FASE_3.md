# Propostas de síntese — Fase 3

Status editorial: **aprovadas pelo usuário após a Fase 3**. Estes textos são descrições de interface, não citações nem reflexões principais. A aprovação permite sua apresentação na caixa de sentimentos durante a Fase 4, sem influência no ranking.

| Combinação direcional | Descrição humana proposta |
| --- | --- |
| Autoconhecimento + Confusão | Talvez compreender a si mesmo esteja difícil porque as dúvidas mudam antes que uma resposta consiga se firmar. |
| Autoconhecimento + Insegurança | Talvez olhar para si esteja esbarrando no receio de não encontrar uma imagem suficientemente segura ou aceitável. |
| Autoconhecimento + Confusão + Insegurança | Talvez você esteja tentando compreender quem é enquanto suas dúvidas exigem uma certeza que nenhuma definição consegue oferecer por completo. |
| Confusão + Autoconhecimento | Pode estar difícil organizar as dúvidas porque cada resposta parece também mudar a maneira como você se compreende. |
| Insegurança + Autoconhecimento | Talvez a dúvida sobre o próprio valor esteja tornando o olhar para si mais exigente do que esclarecedor. |
| Luto + Saudade | Talvez a ausência esteja doendo também porque o vínculo continua vivo na memória e nos gestos que ficaram. |
| Saudade + Luto | A lembrança pode estar trazendo presença e, ao mesmo tempo, a consciência de que algo não pode voltar da mesma forma. |
| Amor + Medo | Talvez o desejo de se aproximar esteja acompanhado pelo receio de perder, ferir-se ou não ser correspondido. |
| Medo + Amor | Talvez o medo esteja ganhando força justamente onde existe algo ou alguém que importa profundamente. |
| Ansiedade + Medo | Pode estar difícil separar o perigo real daquilo que a mente antecipa enquanto tenta se preparar para o que ainda não aconteceu. |
| Medo + Ansiedade | Talvez uma ameaça possível esteja mantendo a mente em alerta, procurando controlar antes mesmo de saber o que acontecerá. |
| Tristeza + Solidão | Talvez a tristeza esteja ficando mais pesada porque parece não haver com quem dividir plenamente o que está sendo vivido. |
| Solidão + Tristeza | Talvez a distância dos outros esteja tornando mais visível uma tristeza que antes encontrava algum abrigo na presença. |
| Raiva + Culpa | Talvez a necessidade de reconhecer um limite esteja se chocando com o receio de ter ferido alguém ou ido longe demais. |
| Culpa + Raiva | Talvez o peso da responsabilidade esteja misturado à raiva pelo que aconteceu, pelo que foi permitido ou pelo que não pôde ser evitado. |
| Falta de propósito + Confusão | Pode estar difícil encontrar direção quando muitas possibilidades existem, mas nenhuma parece ter significado suficiente. |
| Confusão + Falta de propósito | Talvez as dúvidas estejam se acumulando porque ainda não há um sentido claro capaz de organizar as escolhas. |
| Falta de propósito + Insegurança | Talvez a busca por direção esteja esbarrando no medo de escolher errado ou de não ser capaz de sustentar um caminho. |
| Esperança + Luto | Talvez exista uma abertura para continuar, ainda que ela precise conviver com a perda sem apagá-la. |
| Luto + Esperança | Talvez a perda ainda ocupe o centro, enquanto uma possibilidade discreta de continuidade começa a aparecer sem exigir pressa. |
| Amor + Saudade | Talvez o vínculo continue procurando presença por meio da memória, do desejo de proximidade e daquilo que ainda importa. |
| Saudade + Amor | Talvez a falta esteja revelando a profundidade de um afeto que permanece presente mesmo na distância. |

## Fallback cauteloso proposto

> Esses sentimentos podem estar se atravessando de uma maneira difícil de separar. Talvez não seja necessário compreender todos eles de uma vez.

O fallback foi aprovado junto ao primeiro lote para uso cauteloso na interface.
